## 🎯 Features & Testing

### 1. **Authentication System**

#### Sign Up with Role Selection
- Go to http://localhost:5175/login
- Click "Sign Up"
- **NEW**: Choose between **Customer** or **Admin** role
  - 👤 Customer: Regular user who can purchase products
  - 🛡️ Admin: Administrator role for store management
- Fill in: Name, Email, Password, Phone (optional)
- Submit → Receive OTP email (check console in dev mode for OTP)
- Enter OTP on verification screen
- ✅ Account verified and ready to login

#### Login
- Use your registered email and password
- ✅ Receive access token and refresh token
- Redirected to home page as authenticated user

#### Logout
- Click profile avatar → Click logout icon in navbar
- ✅ Session cleared and returned to login

## 📋 API Endpoints

### Authentication
- `POST /api/auth/signup` - Create account
- `POST /api/auth/verify-otp` - Verify email with OTP
- `POST /api/auth/login` - Login user
- `POST /api/auth/logout` - Logout user

### Products
- `GET /api/products` - Get all products (paginated)
- `GET /api/products/:id` - Get product details
- `GET /api/products/featured` - Get featured products
- `GET /api/products/trending` - Get trending products
- `GET /api/products/search` - Search products

## 🛠️ Technical Stack

### Frontend
- **React 19** with TypeScript
- **Vite** (lightning-fast builds)
- **Tailwind CSS** (styling)
- **Framer Motion** (animations)
- **React Query** (data fetching)
- **Zustand** (state management)
- **React Hook Form** + **Zod** (form validation)
- **Axios** (HTTP client)

### Backend
- **Node.js** with Express
- **TypeScript**
- **MongoDB** (database)
- **JWT** (authentication)
- **Nodemailer** (email/OTP)
- **Mongoose** (ODM)
