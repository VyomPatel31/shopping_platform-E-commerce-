## 🎯 Testing Flow

### 1. Make sure servers are running:
```bash
# Terminal 1
cd Backend
npm run dev

# Terminal 2
cd Frontend
npm run dev
```

### 2. Go to: http://localhost:5175

### 3. Test the full flow:
```
1. Click "Login" button
2. Click "Sign Up"
3. Choose "Customer" or "Admin"
4. Fill in details
5. Click "Sign Up"
6. Check console for OTP (dev mode)
7. Enter OTP on verification screen
8. Click "Back to Login"
9. Login with email/password
10. Browse products on homepage
11. Click any product → See full details
12. Adjust quantity → Add to cart
13. See cart count update in navbar
```
